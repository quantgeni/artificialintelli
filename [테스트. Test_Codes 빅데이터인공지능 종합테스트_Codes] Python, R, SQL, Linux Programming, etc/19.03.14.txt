6. 오라클 데이터베이스를 외부에서 접속 가능하도록 방화벽을 설정한다   
[root@OracleDBServer ~]#  vi /etc/firewalld/zones/public.xml
<?xml version="1.0" encoding="utf-8"?>
<zone>
  <short>Public</short>
  <service name="ssh"/>
  <service name="dhcpv6-client"/>
  <port protocol="tcp" port="3306"/>
  <port protocol="tcp" port="1521"/>
</zone>

[root@OracleDBServer ~]# firewall-cmd --reload
success
[root@OracleDBServer ~]#




7. 오라클 데이터베이스를 이용해서 테이블스페이스와 사용자 계정을 생성한다   
     계정명 : oracleuser, 암호 : 987654
(sqlplus 접속 과정 및 connect, resource 권한 부여[grant] 생성 포함. 아래에서 숫자 2, 3은 
직접 입력한 값이 아닌 다음줄 자동 표시된 내역임)


[root@OracleDBServer ~]# . /u01/app/oracle/product/11.2.0/xe/bin/oracle_env.sh
[root@OracleDBServer ~]# su oracle
[oracle@OracleDBServer centos]$ sqlplus /nolog
